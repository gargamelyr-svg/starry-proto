# Bug report #2
**Title:** Dialogue restarts from the beginning after completion

## Steps to reproduce
1. Launch the game
2. Walk point-blank to the NPC
3. Press E to open dialogue
4. Use arrow keys to select an option, press Space to confirm
5. Press E again on the same NPC at the same proximity

## Expected behaviour 
Re-interacting with the NPC shows a new response acknowledging the previous exchange.

## Actual behaviour 
The original dialogue restarts from the beginning as if the conversation never happened.

## Severity
Major - breaks narrative progression in any story-driven context
